# Research Orchestrator

Coordinates topic-driven research.

## Responsibility

- Interpret a user topic.
- Create a `ResearchProject`.
- Create and schedule `ResearchTask` records.
- Coordinate research desks.
- Emit `ActivityEvent` records.
- Produce source-grounded `ResearchArtifact` outputs.

## MVP Bias

Start with deterministic planning for 2-3 research facets before adding more autonomous planning.
