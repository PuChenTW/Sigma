# Thesis Registry

Thesis lifecycle and versioning boundary.

## Responsibility

- Persist thesis records.
- Track thesis status and revisions.
- Link theses to evidence, artifacts, and decisions.
- Preserve assumptions, catalysts, invalidation conditions, confidence, and outcome.

## Statuses

Initial status vocabulary:

- `draft`
- `researching`
- `active`
- `challenged`
- `invalidated`
- `closed`
