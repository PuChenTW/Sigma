# Web App

Frontend for AI Investment Research Studio.

## Responsibility

- Let the user submit research topics.
- Show research project and task status.
- Show thesis detail with evidence references.
- Show Trading Committee proposals.
- Let the user approve or reject proposals.

## Prototype Reference

Use `frontend_prototype/投資工作室.dc.html` as workflow and layout reference:

- Dashboard
- Assign Task
- Research / Thesis Detail
- Decision Desk
- Minimal Positions / Decision Outcome view

Do not treat `frontend_prototype/support.js` as production architecture.
