# Schemas Package

Shared schemas for API payloads, persistence records, and agent IO contracts.

## Responsibility

- Define explicit request and response shapes.
- Keep Trading Committee inputs compact.
- Preserve citation IDs and source references.
- Make invalid states hard to represent.

## Initial Schema Groups

- Research project schemas.
- Research task schemas.
- Evidence and citation schemas.
- Artifact schemas.
- Thesis schemas.
- Committee evaluation schemas.
- Decision proposal and decision schemas.
